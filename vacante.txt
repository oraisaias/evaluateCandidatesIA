🟢 Requisitos Técnicos (Prioridad por nivel de importancia)
1️⃣ Redux & Redux-Saga (9/10 - Crítico)

Dominio en el manejo de estado con Redux y Redux-Saga.

2️⃣ React (Frontend Web - 9/10 - Esencial)

Capacidad para levantar proyectos desde cero con React.

3️⃣ Testing (7/10 - Importante, pero no crítico)

Experiencia en pruebas unitarias e integración.
Uso de herramientas como Jest, Testing Library o Cypress.
4️⃣ Backend y Java (6/10 - Necesario a nivel básico)

No se requiere desarrollo en Java, pero sí conocimientos básicos.

5️⃣ Git y Flujos de Trabajo (8/10 - Importante)

Uso avanzado de Git: ramas, PRs, publicación de cambios.
Familiaridad con flujos de trabajo colaborativos.
